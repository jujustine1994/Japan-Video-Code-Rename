# AV 檔案命名規範

## 標準格式

```
[番號] [女優名(別名)] - [日文片名] [女優名].[副檔名]
```

## 欄位說明

### 番號
- 格式：`大寫字母-數字`（英文全大寫）
- 例：`DDT-435`、`GTJ-065`、`DASD-289`
- 若原始番號為小寫需轉大寫，例 `ddt-435` → `DDT-435`
- 若番號含特殊字元（如空格），需標準化為 `字母-數字` 格式

### 女優名
- 主名在前，別名用半形括號 `()` 包起，例：`新井エリー(晶エリー、大沢佑香)`
- 多位女優以空格分隔，例：`美咲結衣 結城みさ`
- 未知女優統一用：`未知女優`
- 括號內多個別名用頓號 `、` 分隔

### 分隔符
- 固定為 ` - `（前後各一個半形空格）

### 片名
- 使用**日文原名**，不使用中文翻譯或英文翻譯
- 從 javbus.com 查詢取得

### 尾部女優名
- 片名後加一個空格，再列出女優名
- 多位女優：盡量全列，空格分隔
- 單女優：直接重複主名（不含別名括號）
- 若女優太多（超過4位），可省略尾部女優名

### 分集
- 格式：`(1)`、`(2)`（半形括號 + 數字）
- 加在副檔名之前，例：`DDT-153 女優名 - 片名 女優名(1).mp4`
- 若無法辨識集數，保留原始數字標記

## 完整範例

**單女優：**
```
DDT-435 吉田花 - 解禁アナル・FUCK 吉田花.mp4
```

**單女優含別名：**
```

```

**多女優：**
```
DDT-406 美咲結衣 結城みさ - 義母・フィスト奴隷 結城みさ 美咲結衣.mp4
DVDES-543 海宝美羽 今村なつ 春希ゆきの 有村千佳 水嶋あずみ - SEXのハードルが異常に低い世界 3.mp4
```

**未知女優：**
```
DDT-518 未知女優 - TOHJIRO全集 Vol.15 スカ・ウン汁噴射.mp4
```

**分集：**
```
DDT-153 橘未稀(石川麻矢) - 拘束椅子トランス 橘未稀(1).mp4
DDT-153 橘未稀(石川麻矢) - 拘束椅子トランス 橘未稀(2).mp4
```

## 問題檔案類型與處理方式

| 類型 | 範例 | 處理 |
|------|------|------|
| 只有番號 | `DGEN-013 dgen013.mp4` | 查 javbus 補片名和女優 |
| 來自下載站 | `DDT-084 Masochist Trance - HighPorn.mp4` | 查 javbus 重建正確格式 |
| 中文翻譯片名 | `DDT-401 水菜麗 - 強 D 水菜麗在南部島嶼.mp4` | 查 javbus 換回日文原名 |
| 番號小寫 | `dgen 018 - BEST XXX TUBE.mp4` | 標準化番號後查 javbus |
| 無法識別番號 | `1002.mp4`、`GSC.mp4` | 標記為待人工處理 |

## 資料來源

- 主要：javbus.com
- 查詢方式：`https://www.javbus.com/[番號]`（番號大寫）
